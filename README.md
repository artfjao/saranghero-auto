# Saranghero-Auto

A browser extension that automatically captures and types #quoteDisplay text into #quoteInput instantly on saranghero.com.

## How it works

This extension injects a dashboard directly into the web page of saranghero.com/play. Once active, it continuously reads the text from the quote display area and simulates realistic typing into the input box to automate the typing process. You can control the typing speed and monitor game stats right from the injected dashboard.

## Installation Guide

### Desktop Users (Chrome, Edge, Brave)
1. Download the ZIP file from the website.
2. Extract the downloaded ZIP file to a folder on your computer.
3. Open your browser and go to the extensions page (e.g., chrome://extensions, edge://extensions).
4. Enable 'Developer mode' (usually a toggle in the top right corner).
5. Click on 'Load unpacked'.
6. Select the extracted folder containing the extension files.

### Mobile Users (Android)
To use this extension on mobile, you need a Chromium browser that supports extensions. We highly recommend using **Quetta Browser** or **Kiwi Browser**.

1. Download the ZIP file from the website.
2. Install **Quetta** or **Kiwi Browser** from the Google Play Store.
3. Open the browser and go to the Extensions menu (e.g., kiwi://extensions for Kiwi).
4. Enable 'Developer mode'.
5. Tap on 'Load unpacked' or upload from file.
6. Select the downloaded ZIP file or the extracted folder.

## How to Use

1. Open saranghero.com/play in your browser.
2. A floating dashboard will appear automatically on the page.
3. The dashboard will display your stats and controls.
4. The extension will automatically detect the inputs and type for you.
5. Click 'STOP AUTO-TYPE' or 'START AUTO-TYPE' to toggle it manually.

## Q&A

**Q: Is it safe?**
A: Yes, it simulates realistic typing. However, use it at your own risk.

**Q: Does it work on mobile?**
A: Yes, using browsers that support extensions like Quetta or Kiwi.
